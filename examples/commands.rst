===============================================
Available Commands for irc3 at irc.freenode.net
===============================================

.. contents::

irc3.plugins.command
====================

help
----

Show help

    ``!help [<cmd>]``

*Require view permission.*

ping
----

ping/pong

    ``!ping``

*Require admin permission.*

irc3.plugins.search
===================

ddg
---

Search using https://duckduckgo.com/api

    ``!ddg <query>...``

*Require view permission.*

irc3.plugins.uptime
===================

uptime
------

Show uptimes

    ``!uptime``

*Require view permission.*

social
======

mb
--

Post to social networks

    ``!mb [--id=<id>] <message>...``

*Require edit permission.*
